# README: Piscine Web Programming (42 Bangkok)

โปรเจกต์นี้เป็นผลงาน Web Portfolio ที่ผมสร้างขึ้นในช่วง 6 วันของการทำ Piscine ที่ 42 Bangkok ครับ เป็นช่วงเวลาที่ผมได้โฟกัสกับการฝึกทักษะ Web Development แบบเข้มข้น และลองผิดลองถูกด้วยตัวเองตั้งแต่วันแรก

### สิ่งที่ได้เรียนรู้และประสบการณ์
ในช่วง 6 วันนี้ สิ่งที่ผมได้มากกว่าแค่การเขียน Code คือกระบวนการคิดแบบ 42 ครับ ผมได้ฝึกการหาคำตอบด้วยตัวเองจาก Documentation และการแลกเปลี่ยนความรู้กับเพื่อน (Peer Learning) เพื่อแก้ปัญหาเฉพาะหน้าภายใต้เวลาที่จำกัด

ในส่วนของตัวเว็บ ผมเน้นการวางโครงสร้างที่ยืดหยุ่น โดยตั้งใจทำให้ ( Responsive Design ) และการจัดการฐานข้อมูลให้เป็นระบบ เพื่อให้ Code ตัวนี้คลีนพอที่จะเปิดเป็น Open Source ให้คนอื่นนำไปศึกษาต่อได้ในอนาคต

### สรุปทักษะที่ใช้
- การจัดการ Frontend ด้วย Next.js และ Tailwind CSS
- การจัดลำดับความสำคัญของ Feature เพื่อให้งานเสร็จทันตามกำหนด
- การใช้งาน Github ในการทำงานร่วมกันการส่งงานต่างๆ

การมาอยู่ที่นี่ทำให้ผมเข้าใจว่า การเป็น Developer ที่ดีไม่ใช่แค่เขียนโปรแกรมเป็น แต่คือการรู้วิธีที่จะเรียนรู้สิ่งใหม่ๆ ตลอดเวลาครับ
